# Mery Ferrando website

This folder contains a static website for GitHub Pages.

## Upload to GitHub

1. Open `https://github.com/meryferrando/meryferrando.github.io`.
2. Click **Add file** and then **Upload files**.
3. Drag the contents of this folder into the upload area. Upload the files themselves, not the outer `meryferrando-site` folder.
4. Make sure you upload:
   - `index.html`
   - `research.html`
   - `teaching.html`
   - the `assets` folder
5. At the bottom, click **Commit changes**.
6. Wait one or two minutes and open `https://meryferrando.github.io`.

Do not change the custom-domain setting yet. Keep the WordPress site active until the GitHub version has been reviewed.
